Introduction

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small- and large-scale projects.

Python is dynamically-typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.

Hello World

Let's create your first program with Python. There is a common tradition to start coding in a new language. We will create a program Hello World. It is very simple. We will need a function print(). Follow the instruction and run the code.

print('Hello world!')

Congratulations! You have just created your first program!

Commenting

Comments are pieces of code, that are used to explain the logic of code. They help to make code more readable. When the code is executed, Python ignores comments.

Comments in Python start with #.

# This piece prints Hello World print('Hello World!')

Comments can be placed at the end of line as well.

print('Hello world') # This piece prints Hello World
Variables

First of all, variable is container to store data values.

There is no a specific command to create variables in Python. You only need to assign a value to it. Let's see an example.

name = 'Alex' print(name) # Alex

You do not need to declare a variable type. This variable can be changed after they have been created.

x = 10 print(x) # 10 x = 'Hello' print(x) # Hello

If you would like to specify the type of your variable, you can do casting. For instance, if you want it to be string, you can use str() method. There are int() and float() methods to convert variables to integers or float respectively. Of course, there are more methods, but we will discuss them in the future. To check the type of variable we can use type() method.

number = 3 print(type(number)) # class 'int' number = float(3) print(type(number)) # class 'float'

One more thing. Variables can be declared either with single or double quotes

Data Types

Let's learn about basic Data Types in this section. Data type is a super important concept in programming. Variables can store data of different types, and different types can do different things.

  1. str

    2. Strings are sequences of character data.

    name = 'David' print(type(name)) # class 'str'
  2. int

    3. Integer is a whole number, positive or negative, without decimals, of unlimited length.

    age = 23 print(type(age)) # class 'int'
  3. float

    4. The float type in Python designates a floating-point number. float values are specified with a decimal point.

    number = 4.5 print(type(number)) # class 'float'
  4. list

    5. Lists are used to store multiple items in a single variable.

    fruits = ['apple', 'banana', 'orange']
  5. dict

    6. Dictionaries are used to store data values in key:value pairs.

    person = {'name': 'Alex', 'age': 30}
  6. bool

    7. Objects of Boolean type may have one of two values, True or False
